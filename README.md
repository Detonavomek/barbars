# Bot for barbars.ru

## Install

### Pre-installed packages

Next packages should been installed:
```
git
python(version 3.6+)
pip3
```

### Install project

```sh
git clone https://github.com/detonavomek/barbars.git
cd barbars
pip3 install -r requirements.txt
```

## Run

```sh
python3 main.py --config=conf/default.json
```
